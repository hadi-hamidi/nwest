## 🚀 كيفية الاستخدام

### الخطوة 1: تسجيل الدخول كمدير رئيسي
1. اذهب إلى صفحة تسجيل الدخول
2. استخدم:
   - **اسم المستخدم**: `admin`
   - **كلمة المرور**: `admin123`
3. ستتم إعادة توجيهك إلى لوحة الإدارة الرئيسية

### الخطوة 2: إدارة المستخدمين
1. من القائمة الجانبية، انقر على "إدارة المستخدمين"
2. ستظهر قائمة بجميع المستخدمين المسجلين
3. يمكنك:
   - **إضافة مستخدم جديد**: انقر على "إضافة مستخدم جديد"
   - **عرض التفاصيل**: انقر على أيقونة العين
   - **حذف المستخدم**: انقر على أيقونة الحذف

### الخطوة 3: اختبار تسجيل الدخول للمستخدمين الإقليميين
1. اذهب إلى صفحة تسجيل الدخول
2. استخدم بيانات أي من المستخدمين التجريبيين أعلاه
3. ستتم إعادة توجيهك إلى لوحة التحكم المناسبة

## 🔧 التطوير المستقبلي

### المميزات المخططة
1. **تعديل المستخدمين**: إضافة وظيفة تعديل بيانات المستخدمين
2. **إشعارات**: نظام إشعارات للمستخدمين الجدد
3. **تسجيل الأنشطة**: تتبع جميع أنشطة المستخدمين
4. **صلاحيات مخصصة**: إمكانية إنشاء صلاحيات مخصصة
5. **تصدير البيانات**: تصدير قائمة المستخدمين

### التحسينات التقنية
1. **تشفير كلمات المرور**: استخدام bcrypt أو مكتبة مشابهة
2. **قاعدة بيانات**: الانتقال من localStorage إلى قاعدة بيانات حقيقية
3. **API**: إنشاء API للتعامل مع المستخدمين
4. **المصادقة**: نظام مصادقة متقدم

## 🚨 ملاحظات مهمة
- النظام يعمل حالياً مع localStorage

- يجب إضافة نظام تسجيل الأنشطة
